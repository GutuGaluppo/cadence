# Privacy Policy

Cadence is designed as a local-first desktop product.

## What the app stores

- task data in a local SQLite database on the device
- timer and interface preferences in local browser storage used by the desktop shell

## What the app does not require

- no account
- no mandatory sign-in
- no cloud sync in the current version
- no bundled analytics or ad tracking scripts in this repository

## Network behavior

The desktop app is intended to work without a user account or a remote backend for core functionality.

If you open external links such as GitHub Releases or the public landing page, those third-party services may process standard request metadata such as IP address, user agent, and access logs under their own policies.

## Public website

The `site/` landing page in this repository is static. It links to GitHub resources for downloads, release notes, and issue reporting.

If you publish that site through GitHub Pages or another host, the hosting provider may keep standard web server logs. Review the hosting provider’s own terms before launch.

## Support requests

If you open a public issue, do not include secrets, private credentials, license keys, or personal data that should not be publicly visible.

## Changes

This file should be reviewed before every public release that changes storage, networking, telemetry, or third-party integrations.
